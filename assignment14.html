SQL

Upon completing all Assignment 14 activities, I learned a ton of information about SQL. SQL stands for Structured Query language. SQL is used as a database programing language originally developed by Donald D. chamberlain, and Raymond F. Boyce at IBM in the early 1970s. SQL can be used for data inserts, query data and retrieving data, update and delete, and general data access. For example on a site where a product is bought, the process of customer buying a product is repeated, SQL makes it so that the whole procedure/ code can be reused. Some common commands/terms are Select, From, Join, Where, and Order By. These commands are used to navigate through the databases that are created.

Here are some examples of basic SQL commands

  • SELECT * FROM Customers;
  • SELECT CustomerName,City FROM Customers;
  • SELECT COUNT(CustomerID), Country FROM Customers GROUP BY Country;
  • SELECT TOP 3 * FROM Customers;
